# Simple Banking Application (Core Python)

## Overview
This is a menu-driven banking application developed using Core Python. The program simulates basic banking operations using file-based storage.

## Features
- Account creation
- Deposit money
- Withdraw money
- Balance enquiry
- Transaction tracking
- File handling
- Exception handling

## Technologies Used
- Python

## How to Run
1. Download or clone this repository.
2. Open the project folder.
3. Run the main file:
   python app.py

## Project Structure
- app.py – Main program
- bank.py – Banking logic
- accounts.txt – Account data storage
- transactions.txt – Transaction records
